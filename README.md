## About The Project

Basic concepts of image, image processing and machine vision. Image segmentation. Representation and description of images. Context recognition. Image retrieval. Automatic image annotation. Object description and recognition. You can install all the dependencies from `labenv.txt`.

### My projects

* [Image Clasification CIFAR10](https://github.com/vasetrendafilov/MachineVision/blob/main/Clasification/imgclassificationcifar10.ipynb) using VGG model.
* [FCN](https://github.com/vasetrendafilov/MachineVision/blob/main/Localization/fcn.ipynb) with simple VGG model substructure and made a jupyter notebook for better visualization.
* [Segmentation](https://github.com/vasetrendafilov/MachineVision/blob/main/Segmentation/semantic_seg_train.py) with unet used on segmenting out cells.

### Built With

* [OpenCV](https://opencv.org/)
* [Numpy](https://numpy.org/)
* [Keras](https://keras.io/)
